This package includes programs and dataset.

The programs include EsPa-DSGD, BaPa-DSGD, EsPa-CCD and BaPa-CCD

The datasets include MovieLens 100K, 1M, 10M, 20M, Netflix and Netflix2. They are stored by (row indice | colume indice | value). Each dataset in folder "transpose" corresponds to the transpose of the dataset in  "original".

Each program prints the RMSE of each iteration. When iteration stops, it prints the number of iterations and average elapsed time of each iteration. The result is stored in record.txt.
